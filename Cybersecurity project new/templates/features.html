Scanner Modules

Features of the Assessment Tool

Header Analysis

Checks for security headers that help reduce clickjacking, MIME sniffing, and script injection risk.

Port Scanning

Uses socket connections to check common ports and highlight unnecessary exposure on the target host.

XSS Detection

Submits a harmless reflection payload to forms and checks whether it is returned in the response.

SQL Injection Detection

Uses a safe demo payload to look for SQL error messages or suspicious database-related responses.

Risk Classification

Assigns Low, Medium, High, or Critical severity to help present findings clearly during review.

Report Generation

Builds a structured scan summary that can later be extended for PDF export or database storage.
